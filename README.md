# Setup

```
make install
vim +PlugInstall +qall
```

# Mappings

```
Buffers            <leader>+b
Code               <leader>+C
Config             <leader>+c
Mode               <leader>+m
Projects           <leader>+p
Run                <leader>+r
Splits             <leader>+s
Tabs               <leader>+t
VCS                <leader>+v
```

Use `<leader>` key to see mappings, default `<space>`.

Use `?` to see the cheat sheet.
